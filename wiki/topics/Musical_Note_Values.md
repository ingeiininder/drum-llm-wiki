# Musical Note Values
Musical note values represent the duration of a sound to be produced on an instrument. In [[Drum Reading]], understanding these values is fundamental to accurately interpreting and performing rhythms. Each note has a specific duration relative to the beat and the time signature.

The common note values, as presented in Unit 1, include:
*   [[Whole Note]] (4 beats in 4/4 time)
*   [[Half Note]] (2 beats in 4/4 time)
*   [[Quarter Note]] (1 beat in 4/4 time)
*   [[Eighth Note]] (1/2 beat in 4/4 time)
*   [[16th Note]] (1/4 beat in 4/4 time)

These values are crucial for constructing [[Rhythmic Groupings]] and understanding the rhythmic framework of any piece of music.

**Pro-Tip**: To truly internalize note values, practice subdividing them verbally (e.g., "1-e-&-a" for 16th notes) while tapping or playing. This helps build a strong internal clock and ensures accurate placement of notes within the beat.
