# Musical Note Values
Musical note values represent the [[Duration|duration]] of a sound to be produced on an instrument. In [[Drum_Reading]], understanding these values is fundamental to accurately interpreting and performing rhythms. Each note has a specific [[Duration|duration]] relative to the [[Beat|beat]] and the [[Time_Signature|time signature]].

The common note values, as presented in Unit 1, include:
*   [[Whole_Note]] (4 beats in 4/4 time)
*   [[Half_Note]] (2 beats in 4/4 time)
*   [[Quarter_Note]] (1 [[Beat|beat]] in 4/4 time)
*   [[Eighth_Note]] (1/2 [[Beat|beat]] in 4/4 time)
*   [[16th_Note]] (1/4 [[Beat|beat]] in 4/4 time)

These values are crucial for constructing [[Rhythmic_Groupings]] and understanding the rhythmic framework of any piece of music.

**Pro-Tip**: To truly internalize note values, practice [[Subdividing|subdividing]] them verbally (e.g., "1-e-&-a" for [[16th_Notes|16th notes]]) while tapping or playing. This helps build a strong internal clock and ensures accurate placement of notes within the [[Beat|beat]].
