# Dotted Half Notes (Practical Use)
[[Dotted_Half_Note]]s, which are worth three [[Beat]]s, are frequently used in musical notation. However, their placement in [[4-4_Time_Signature]] can sometimes conflict with the strict [[2_3_Rule]].

**Common Scenarios:**
1.  **Dotted Half Note on Beat 1**: If a dotted half note starts on beat 1, it sustains through beats 1, 2, and 3. This technically "obscures" the [[Downbeat]] of beat 3, as a single note spans across it. Despite this, it is often accepted as a "rule breaker" in many musical scenarios due to its common usage and clear rhythmic intent.
    *   *Theoretical Correction*: To strictly adhere to the 2/3 rule, this rhythm would ideally be written as a [[Half_Note]] on beat 1 tied to a [[Quarter_Note]] on beat 3. This explicitly marks the downbeat of 3.
2.  **Dotted Half Note on Beat 2**: If a dotted half note starts on beat 2, it sustains through beats 2, 3, and 4. This also obscures the downbeat of beat 3.
    *   *Theoretical Correction*: This would ideally be written as a [[Quarter_Note]] on beat 2 tied to a [[Half_Note]] on beat 3.

**Pro-Tip:** While understanding the theoretical "correctness" is important, drummers must be pragmatic. If a dotted half note is clearly written and its rhythmic intent is unambiguous, play it as written. However, be aware of the underlying principles, especially when [[Sight_Reading]], as it helps in interpreting potentially ambiguous notation. The goal is always clear communication and accurate performance.
