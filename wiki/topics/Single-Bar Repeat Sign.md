# Single-Bar Repeat Sign
A [[Single-Bar Repeat Sign]] is a common notation device used to indicate that the preceding measure should be repeated. It serves as a shorthand to create cleaner, more concise musical charts, especially in genres like jazz, pop, and rock where rhythmic patterns often repeat for extended periods.

**Symbol:**
The symbol for a single-bar repeat is a forward slash (/) with two dots, one above and one below the middle line of the staff.

**Placement:**
This symbol sits directly in the middle of the measure it is replacing.

**Usage:**
Instead of writing out the same measure multiple times, a composer can write the initial measure and then use single-bar repeat signs for subsequent repetitions. For example, if a [[Drum Groove]] repeats for four measures, the first measure is written out, and the next three measures would contain a single-bar repeat sign.

**Professional Insight:**
While single-bar repeats simplify notation, drummers must remain vigilant. It's easy to lose track of how many times a measure needs to be repeated, especially in longer sections.
*   **Counting is Key:** Always count the number of measures you've played and how many repeats are indicated.
*   **Context Matters:** Pay attention to the musical context. If a repeated series carries over to a new line, the rhythm is often notated again at the beginning of the new line to reassure the player.
*   **Combining Repeats:** Single-bar repeats can also be used in conjunction with other [[Repeat Signs]] like [[Start Repeat Sign]] and [[End Repeat Sign]] to create longer repeated sections, requiring careful tracking of the musical form.
