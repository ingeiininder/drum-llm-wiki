# Flam Rudiment Applications

This unit integrates three primary flam-based [[Rudiments]] into a drumset context, utilizing a standard [[Bass Drum]] and [[Hi-Hat]] ostinato.

### Core Rudiments Covered
- **[[Flam Accent]]**: A triplet-based rudiment where the first note of each group is flammed.
- **[[Flam Tap]]**: A rudiment consisting of a flam followed by a single tap of the same hand (rR L lL R).
- **[[Flammed Mill]]**: An inverted paradiddle with a flam on the first note of each group (rR L R R lL R L L).

### Execution Rules
- **No Limb Flams**: A common error is "flamming" the snare drum note against the bass drum or hi-hat. The primary note of the flam must land exactly with the foot pattern.
- **Sticking Reversal**: Note that the **Flam Accent** (Ex 4.1) does not naturally reverse sticking. You must consciously practice leading with both the right and left hands.
- **Tempo**: Maintain a slow tempo (40-80 BPM) until the "grace note" of the flam is consistently tight and the primary note is perfectly synchronized with the [[Ostinato]].
