# Note Length Interpretation
In drum set notation, **Note Length** dictates the [[Orchestration]] and technique used to execute an [[Ensemble_Figures]]. While a practice pad produces a uniform, short decay, the drum set allows for a wide spectrum of sustain.

### Long Notes vs. Short Notes
*   **Long Notes**: Any note rhythmically lasting for more than one [[Beat|beat]] (e.g., dotted [[Quarter_Notes|quarter notes]], half notes, or tied notes). These are typically voiced on **Crash Cymbals** to utilize their natural sustain.
*   **Short Notes**: Notes lasting one [[Beat|beat]] or less. These require "dry" sound sources or specific dampening techniques.

### Technical Execution
1.  **Cymbal Chokes**: To indicate a specific end to a note (e.g., a half note ending on [[Beat|beat]] 3), the drummer must "choke" or grab the cymbal on the rest.
2.  **[[Subdivision|Subdivision]]**: Accurate note length is impossible without internalizing the underlying eighth or sixteenth-note grid. 

### Pro-Tip
Always look ahead in the [[Chart|chart]]. If you see a long note followed immediately by a rest, prepare your hand to choke the cymbal. If the note is "short" (like a [[Staccato|staccato]] eighth note), consider using the [[Snare_Drum]] or a [[Choked_Cymbal]] hit.
