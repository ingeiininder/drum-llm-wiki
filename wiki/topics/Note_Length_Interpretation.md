# Note Length Interpretation
In drum set notation, **Note Length** dictates the [[Orchestration]] and technique used to execute an [[Ensemble Figure]]. While a practice pad produces a uniform, short decay, the drum set allows for a wide spectrum of sustain.

### Long Notes vs. Short Notes
*   **Long Notes**: Any note rhythmically lasting for more than one beat (e.g., dotted quarter notes, half notes, or tied notes). These are typically voiced on **Crash Cymbals** to utilize their natural sustain.
*   **Short Notes**: Notes lasting one beat or less. These require "dry" sound sources or specific dampening techniques.

### Technical Execution
1.  **Cymbal Chokes**: To indicate a specific end to a note (e.g., a half note ending on beat 3), the drummer must "choke" or grab the cymbal on the rest.
2.  **Subdivision**: Accurate note length is impossible without internalizing the underlying eighth or sixteenth-note grid. 

### Pro-Tip
Always look ahead in the chart. If you see a long note followed immediately by a rest, prepare your hand to choke the cymbal. If the note is "short" (like a staccato eighth note), consider using the [[Snare Drum]] or a [[Choked Cymbal]] hit.
