# Metric Modulation

**Metric Modulation** is a technique where a change in feel, meter, or tempo is derived from a specific note value encountered in the previous section of music. Unlike a simple tempo change, metric modulation creates a mathematical link between the "old" tempo and the "new" tempo.

### Simple vs. Complex Modulations
*   **Simple Modulations**: These involve common shifts like changing the beat unit from a quarter note to an eighth note (effectively doubling or halving the tempo).
*   **Practical Application**: Composers use these to change feels within a chart in the clearest manner possible.

### Value Indicators
A [[Value Indicator]] is a notation (e.g., $q = e$ or $q = h$) that tells the performer how the previous pulse relates to the new one.
*   **Example**: If a chart modulates from quarter notes at 165 BPM to a section where the previous quarter note now equals an eighth note, the new tempo becomes 82.5 BPM.

### Stylistic Influence
Notation heavily influences how a [[Groove]] is perceived. For example, [[Jazz Swing]] is traditionally written in 4/4. Writing it in a way that forces the hi-hat onto "upbeats" (mathematically correct but stylistically wrong) results in a "stiff" and "flat" performance.
