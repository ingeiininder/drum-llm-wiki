# Metric Modulation

**Metric Modulation** is a technique where a change in feel, [[Meter|meter]], or [[Tempo|tempo]] is derived from a specific [[Note_Value|note value]] encountered in the previous section of music. Unlike a simple [[Tempo|tempo]] change, metric modulation creates a mathematical link between the "old" [[Tempo|tempo]] and the "new" [[Tempo|tempo]].

### Simple vs. Complex Modulations
*   **Simple Modulations**: These involve common shifts like changing the [[Beat|beat]] unit from a quarter note to an eighth note (effectively doubling or halving the [[Tempo|tempo]]).
*   **Practical Application**: Composers use these to change feels within a [[Chart|chart]] in the clearest manner possible.

### Value Indicators
A [[Value_Indicator]] is a notation (e.g., $q = e$ or $q = h$) that tells the performer how the previous [[Pulse|pulse]] relates to the new one.
*   **Example**: If a [[Chart|chart]] modulates from [[Quarter_Notes|quarter notes]] at 165 BPM to a section where the previous quarter note now equals an eighth note, the new [[Tempo|tempo]] becomes 82.5 BPM.

### Stylistic Influence
Notation heavily influences how a [[Groove]] is perceived. For example, [[Jazz_Swing]] is traditionally written in 4/4. Writing it in a way that forces the [[Hi-Hat|hi-hat]] onto "upbeats" (mathematically correct but stylistically wrong) results in a "stiff" and "flat" performance.
