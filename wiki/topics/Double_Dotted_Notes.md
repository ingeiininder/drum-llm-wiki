# Double Dotted Notes

**Double dotted notes** are an advanced form of rhythmic notation. While less common than single [[Dotted Notes]], they are essential for reading complex orchestral or contemporary drum charts.

### The Rule of the Second Dot
The first dot adds **50%** of the original note's value. The second dot adds **50% of the first dot's value** (or 25% of the original note).

### Mathematical Examples
1. **Double Dotted Half Note**:
   - Half Note = 2 beats.
   - 1st Dot = 1 beat (half of 2).
   - 2nd Dot = 1/2 beat (half of 1).
   - **Total** = 3.5 beats.
2. **Double Dotted Quarter Note**:
   - Quarter Note = 1 beat.
   - 1st Dot = 1/2 beat.
   - 2nd Dot = 1/4 beat.
   - **Total** = 1 3/4 beats.
3. **Double Dotted Eighth Note**:
   - Eighth Note = 1/2 beat (4/32).
   - 1st Dot = 1/4 beat (2/32).
   - 2nd Dot = 1/8 beat (1/32).
   - **Total** = 7/32 of a beat.

### Rhythmic Table
| Note Type | Total Value (Beats) |
| :--- | :--- |
| Double Dotted Whole | 7 beats |
| Double Dotted Half | 3.5 beats |
| Double Dotted Quarter | 1.75 beats |
| Double Dotted Eighth | 0.875 beats |

### Practical Use
In drum set charts, double dots are often used to create very specific, elongated syncopations that resolve on the final 32nd note of a beat.
