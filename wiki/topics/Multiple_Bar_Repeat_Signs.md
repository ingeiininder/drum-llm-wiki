# Multiple-Bar Repeat Signs
[[Multiple_Bar_Repeat_Signs]] are a common notation shorthand used in musical charts to indicate that a specific number of previous bars should be repeated. This saves space and simplifies [[Drum Chart Reading]].

**Function**:
*   Similar to a single-bar repeat sign, but it instructs you to repeat a *certain amount* of previous bars, not just one.
*   A modified bar repeat symbol is used, with the exact number of measures to be repeated placed above the symbol (e.g., a "2" for a two-bar repeat, a "4" for a four-bar repeat).

**Important Interpretation**:
*   **Correct**: Repeat the *previous amount of bars indicated* by the number above the symbol.
*   **Common Mistake**: Do *not* repeat the previous bar by the *amount of times* indicated. For example, a "2" above the symbol means repeat the *previous two bars*, not repeat the previous bar twice.

**Pro-Tip**: Always double-check the number above the repeat sign. Misinterpreting this symbol can lead to significant errors in musical form and timing. When practicing, mentally or physically mark the beginning and end of the phrase being repeated to ensure accuracy.
