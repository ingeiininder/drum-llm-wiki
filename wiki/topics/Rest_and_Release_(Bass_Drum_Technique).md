# Rest and Release ([[Bass_Drum_Technique|Bass Drum Technique]]) (低音鼓休息與釋放技巧)
一種低音鼓技巧，腳和槌頭在擊打之間都休息和釋放。這對於控制低音鼓槌頭、避免 [[Felt-Flutter_(Bass_Drum)]] 以及讓鼓完全共鳴至關重要，尤其是在執行 [[Off_the_Head_Stroke_(Bass_Drum)]] 時。這與「埋入式」方法不同，因為在擊打之間不能將腿保持在「攻擊」位置。
