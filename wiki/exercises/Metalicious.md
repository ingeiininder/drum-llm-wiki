# Metalicious

"Metalicious" is a high-intensity [[Metal]] drum study written by Steve Kilgallon. It focuses on [[Double Bass Drumming]] patterns, unison hits, and navigating a complex "Road Map" with guitar solos and riffs.

### Key Features
- **The "Chainsaw" Sound**: A metaphor for the aggressive, consistent double-pedal drive required for the track.
- **Unison Hits**: Beat 4 frequently features a simultaneous hit of the snare, kick, and cymbal. This is a primary tool for "heavying up" the groove.
- **Sectional Contrast**: 
    - *Groove A*: The primary driving feel.
    - *Groove B*: Described as sounding like "Jaws is coming to eat you," implying a more menacing, perhaps half-time or syncopated feel.

### Performance Notes
- **Empty Space**: The track features deliberate gaps between kick patterns. Drummers are encouraged to use this space for creative fills or to let the guitar riffs breathe.
- **Road Map Navigation**: The song structure includes "Guitar Riffs," "Guitar Solos," and "Drums Drop" sections. Precise counting is required to transition between Groove A and Groove B.

### Source Context
Excerpted from *Double Pedal Metal* by Steve Kilgallon (Hudson Music).
